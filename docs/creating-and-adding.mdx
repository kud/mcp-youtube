## `create-playlist`

Creates a new playlist. Costs **50 quota units**.

| Parameter       | Type                                  | Default     | Notes               |
| --------------- | ------------------------------------- | ----------- | ------------------- |
| `title`         | `string`                              | —           | Required            |
| `description`   | `string`                              | `""`        |                     |
| `privacyStatus` | `"private" \| "unlisted" \| "public"` | `"private"` | Playlist visibility |

Returns the created playlist's `playlistId`, `title`, `description`, `itemCount`, and `privacyStatus`.

## `add-to-playlist`

Adds one or more videos to an existing playlist. Costs **50 quota units per video** — adding ten videos costs 500 units, so batch thoughtfully.

| Parameter    | Type       | Default | Notes                        |
| ------------ | ---------- | ------- | ---------------------------- |
| `playlistId` | `string`   | —       | Required                     |
| `videoIds`   | `string[]` | —       | Required, at least one video |

Videos are added one at a time; if any individual add fails, it's collected into a `failed` list rather than aborting the whole batch. The tool only reports a hard error if _every_ video failed. On partial or full success it returns:

```json
{
  "playlistId": "PLxxxxxxxx",
  "added": ["videoId1", "videoId2"],
  "failed": [],
  "quotaCost": 100
}
```

`quotaCost` reflects only the videos that actually succeeded (`added.length * 50`).

## `update-playlist`

Renames a playlist, changes its description, or changes its privacy. Costs **50 quota units**.

This is a **full-replace** operation on the `snippet`/`status` parts — to avoid silently blanking fields you didn't mean to touch, the tool reads the playlist's current title/description/privacy first, then only overrides the fields you pass.

| Parameter       | Type                                  | Default | Notes                |
| --------------- | ------------------------------------- | ------- | -------------------- |
| `playlistId`    | `string`                              | —       | Required             |
| `title`         | `string`                              | —       | Unchanged if omitted |
| `description`   | `string`                              | —       | Unchanged if omitted |
| `privacyStatus` | `"private" \| "unlisted" \| "public"` | —       | Unchanged if omitted |

## `update-playlist-item`

Moves an item to a new position within a playlist (reorder). Costs **50 quota units**.

| Parameter        | Type     | Default | Notes                              |
| ---------------- | -------- | ------- | ---------------------------------- |
| `playlistItemId` | `string` | —       | Required — the item to move        |
| `playlistId`     | `string` | —       | Required                           |
| `videoId`        | `string` | —       | Required                           |
| `position`       | `number` | —       | Required — new zero-based position |
